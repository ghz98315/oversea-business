# SilkBridge 视觉风格定义

## 一、整体感觉

**关键词**：呼吸感 · 友好 · 专业 · 不拥挤

**风格参考**：Mercury（深蓝+留白+专业感）+ Basecamp（个人化表达+信任建设）

**核心气质**："一个专业的人在跟你平等对话"，不是"一个公司在向你推销"

- 大量留白，区块之间有充足的间距（80-120px）
- 内容不贴边，最大宽度1200px居中
- 文字行间距宽松（1.6-1.8倍）
- 每个区块像一张独立的卡片/场景，滚动时有节奏感
- Hero区：Mercury风格，深蓝渐变，大标题，简洁有力
- 信任区块：Basecamp风格，真人照片+第一人称叙述，像朋友在说话
- CTA按钮：暖橙色，在深蓝/白色背景上都醒目

---

## 二、区块颜色交替方案

用浅色背景交替来区分区块，不用强烈的色块，保持呼吸感：

```
区块1  Hero          → 深蓝渐变背景（深色，压住场面）+ 白色文字
区块2  痛点          → 白色背景（留白呼吸）
区块3  服务          → 浅灰背景（#F8FAFB，微微区分）
区块4  Why Guangdong → 白色背景
区块5  产业带品类     → 浅暖灰背景（#FAFAF8，带一点暖调）
区块6  合作流程      → 白色背景
区块7  信任背书      → 浅蓝灰背景（#F0F4F8，轻微强调）
区块8  FAQ           → 白色背景
区块9  底部CTA       → 深蓝背景（呼应Hero，收尾有力）
```

**规律**：白 → 浅灰 → 白 → 浅暖灰 → 白 → 浅蓝灰 → 白 → 深蓝
- 首尾深色"夹住"整个页面，中间用极浅的灰色交替
- 不是强烈的色块切换，而是微妙的明暗呼吸

## 三、配色系统

### 主色

| 用途 | 色值 | 说明 |
|------|------|------|
| 深蓝（主色） | #1E3A5F | Hero背景、底部CTA、标题强调 |
| 中蓝 | #2D5F8A | 次级标题、链接hover |
| 暖橙（强调色） | #E8853D | CTA按钮、重点标注、图标 |
| 暖橙hover | #D4732E | 按钮hover状态 |

### 中性色

| 用途 | 色值 | 说明 |
|------|------|------|
| 正文黑 | #1A1A2E | 主要文字（不用纯黑，柔和一点） |
| 次要文字 | #5A6B7A | 副标题、描述文字 |
| 浅灰背景 | #F8FAFB | 交替区块背景 |
| 浅暖灰背景 | #FAFAF8 | 交替区块背景（暖调） |
| 浅蓝灰背景 | #F0F4F8 | 信任区块背景 |
| 边框灰 | #E2E8F0 | 卡片边框、分割线 |
| 白色 | #FFFFFF | 主背景 |

## 五、间距与呼吸感

| 元素 | 间距 |
|------|------|
| 区块之间 | 96-120px（大呼吸） |
| 区块内标题到内容 | 40-48px |
| 卡片之间 | 24-32px |
| 段落之间 | 24px |
| 内容最大宽度 | 1200px |
| 内容左右padding | 24px（移动端）/ 48px（桌面端） |

## 六、卡片风格

```
┌─────────────────────────────┐
│                             │  ← 圆角 12-16px
│   内容                      │  ← 内边距 32px
│                             │  ← 无边框 或 1px #E2E8F0
│                             │  ← 轻微阴影 (0 2px 8px rgba(0,0,0,0.04))
└─────────────────────────────┘
```

- 不用重阴影，用极轻的阴影 + 细边框
- hover时阴影微微加深（交互反馈）
- 圆角统一12px或16px

---

## 七、CTA按钮风格

```
主按钮（Get a Free Quote）：
┌──────────────────────────┐
│   Get a Free Quote       │  ← 背景：暖橙 #E8853D
│                          │  ← 文字：白色
└──────────────────────────┘  ← 圆角：8px
                              ← padding：16px 32px
                              ← hover：#D4732E + 微上移2px

次按钮（Read More / Source Tools →）：
文字链接样式，暖橙色 + 箭头，无背景
hover时下划线
```

---

## 八、图标风格

- 使用线性图标（Lucide Icons 或 Heroicons outline）
- 不用填充图标，保持轻盈感
- 图标颜色：深蓝主色 或 暖橙强调色
- 大小：24-32px

---

## 九、动效（轻量）

| 效果 | 说明 |
|------|------|
| 滚动渐入 | 区块进入视口时从下方淡入（fade-up），距离16px，时长0.5s |
| 按钮hover | 微上移2px + 阴影加深 |
| 卡片hover | 阴影加深 + 边框色变化 |
| FAQ展开 | 平滑展开/收起，0.3s |

不做花哨动画，只做"让页面感觉活着"的微动效。
