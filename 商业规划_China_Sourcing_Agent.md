# 商业规划：China Sourcing Agent 服务

## 一、项目定位

**一句话定义**：帮海外中小买家从中国工厂采购产品，赚服务费/佣金。

**阶段规划**：
- 第一阶段（Month 1）：用服务跑通第一单，验证模式
- 第二阶段（Month 3-6）：沉淀重复流程，产品化为SaaS工具

---

## 二、创始人优势

| 维度 | 具体内容 |
|------|---------|
| 行业背景 | 前比亚迪工厂技术经理 |
| 核心能力 | 工厂流程管理、质量管理、供应链管理 |
| 技术能力 | AI coding，能独立开发网站、小程序、App |
| 品类专长 | 消费电子、五金/CNC机械加工件 |
| 供应商资源 | 10+家可直接联系的工厂 |
| 英语能力 | 读写可以，口语一般，可用AI辅助 |
| 启动资金 | 5000元以内 |

## 三、商业模式

### 目标客户
- 海外中小买家（Etsy卖家、亚马逊小卖家、DTC品牌、小型硬件公司）
- 采购金额：$500-$10,000/单
- 痛点：不知道哪个中国工厂靠谱，怕被坑，没有质检能力

### 收入模型

| 收入方式 | 说明 | 举例 |
|---------|------|------|
| 服务费 | 按订单金额收10-15%代采费 | $2000订单收$200-300 |
| 差价 | 工厂报价加15-30%报给客户 | 工厂报$5/件，你报$6.5/件 |
| 验厂/QC费 | 单次验货$150-300 | 出货前验货 |

### 一个月目标
- 成交1-3单
- 收入$500-$2000
- 验证模式可行

## 四、核心差异化（vs Alibaba供应商）

1. **你是人，不是店铺** — 有名字、有脸、有LinkedIn、有BYD背景
2. **你站在买家这边** — Alibaba供应商是卖方，你是买方的代理人
3. **你懂质量** — 能帮客户做供应商筛选和质检
4. **你能比价** — 同一产品问3家工厂，给客户最优方案

---

## 五、信任建设策略

### 网站信任要素
- 真人身份：真实照片、名字、LinkedIn链接
- BYD背景突出："Former Technical Manager at BYD, 8+ years in manufacturing"
- 工厂实拍：产线、质检、仓库照片/视频
- 具体承诺："Get CNC quotes from 3 verified factories within 48 hours"
- 流程展示：模拟完整询价流程showcase

### 网站外信任建设
- LinkedIn个人品牌（每周1-2条专业内容）
- Reddit/论坛专业回答（r/sourcing, r/FBA）
- 自我介绍视频（真实感 > 流利度）

### 交易结构降低信任门槛
- 免费询价（第一次合作零风险）
- 小单起步（$500 trial order）
- PayPal付款（买家有180天争议期保护）
- 全程透明（工厂确认→生产进度→验货照片→物流单号）
- 先验货再发货

## 六、30天执行路线图

### Week 1（Day 1-7）：建基础设施

| 任务 | 具体内容 |
|------|---------|
| 建网站 | 英文sourcing服务网站（Next.js + Tailwind） |

| 渠道 | 做法 |

## 八、风险与应对

| 风险 | 应对 |
|------|------|
| 一个月内没有成交 | 正常，继续积累内容和询盘，延长到2个月 |
| 客户不信任新网站 | 用免费询价+小单+PayPal保护降低门槛 |
| 工厂配合度低 | 小单可能被工厂忽视，选择对小单友好的工厂 |
| 价格竞争 | 不拼价格，拼服务（质检、沟通、透明度） |
| 英语沟通障碍 | AI辅助翻译+邮件为主+模板化沟通 |
